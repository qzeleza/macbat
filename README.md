# MACBAT: Проверка батареи на маке

---

### Скрипт предназначен для оперативного оповещения в Телеграмме о текущем статусе батареи

---

Данный скрипт позволяет решить следующие задачи:
1. Вести подсчет текущего заряда батареи с заданным интервалом времени (задается при запуске, по умолчанию - 1 мин.)
2. Своевременно информировать в Телеграме о достижении минимальных и максимально установленных порогов заряда батареии
3. Информировать по текущему состоянию заряда батареи ноутбука (сообщение выводится после сообщения о достижение порога)

## Структура проекта

└──macbat/
     ├── macbat   		# Основной скрипт запуска и исполения комманд для работы 
     ├── .conf			# Файл где хранятся введенные данные Телеграмм token и id 
     ├── install		# Файл для установки скрипта в систему
     ├── uninstall		# Файл для удаления скрипта из системы
     └── README.md		# Настоящий файл

## Установка скрипта
1. Для установки скрипта склонируйте репозиторий
2. Создайте совего телеграмм бота Создайте в папке проекта файл conf.env и запишите значения token Вашего робота и id диалога
3. Запустите файл установки install

```
git clone адрес_репо
cd ./batch && install
```

## Удаление скрипта
1. Зайдите в папку проекта и запустите скрипт uninstall
2. Удалите папку проекта 

```
cd <путь_клонирования>/batch
./uninstall
```
